# レンタルルーム予約管理システム v2（店舗対応版）

複数店舗・店舗ごとの清掃時間・受付アカウントに対応した予約管理システム。

## 主な機能

- 店舗を動的に追加・削除（管理画面のボタン操作のみ）
- 店舗ごとに清掃時間（インターバル）を設定（コース時間に自動加算）
- 店舗ごとの受付アカウント（受付は自店舗だけ表示・操作）
- コース時間: 20/30/45/60/75/90/120分 ＋ 手入力
- 5分刻みグリッド
- 現在時刻ライン＋今の時刻へ自動スクロール
- 予約の延長（10分刻みボタン＋手入力、延長済み表示、競合時は電話案内）
- 予約時間ずらし（早着・遅れ対応、変更済み表示、競合時は電話案内、店舗へ電話案内）
- 名前（A/B/C コード）またはお客様名の手入力
- 名前の予約画面、自分の予約のみ表示・店舗フィルタ

## 清掃時間の仕組み

コース時間に店舗の清掃時間が自動加算され、その分だけ部屋が占有されます。
例: フィジー（清掃5分）で60分コース → 部屋は65分間ブロック。

## 延長・時間ずらしの仕組み

- 延長: コース時間に追加。次の予約とぶつかる場合は「お電話で調整してください」と表示。
- 時間ずらし: 開始時刻を前後に変更（5分刻み）。他予約と重なる場合は不可。変更後は店舗へ電話連絡を促す。
- 延長・変更した予約は一覧にバッジ（延長済 / 時間変更）が付く。
