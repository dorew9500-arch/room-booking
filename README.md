# レンタルルーム予約管理システム v2（店舗対応版）

複数店舗・店舗ごとの清掃時間・受付アカウントに対応した予約管理システム。

## 主な機能

- 店舗を動的に追加・削除（管理画面のボタン操作のみ、サーバー作業不要）
- 店舗ごとに清掃時間（インターバル）を設定（コース時間に自動加算）
- 店舗ごとの受付アカウント（受付は自店舗だけ表示・操作）
- コース時間: 20/30/45/60/75/90/120分 ＋ 手入力
- 受付メイン画面（左に時系列リスト・右に空きグリッド）
- 名前（A/B/C コード）またはお客様名の手入力
- 名前（提携先）の予約画面、自分の予約のみ表示

## 清掃時間の仕組み

コース時間に店舗の清掃時間が自動で加算され、その分だけ部屋が占有されます。
例: フィジー（清掃5分）で60分コース → 部屋は65分間ブロック（次の予約は65分後から）
